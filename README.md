# 📝 Task Tracker (NotesKeeper)

A clean and responsive React app to add, delete, and search notes — built using **Vite + React** with **localStorage**.

## ✨ Features
- Add/Delete Notes
- Search notes live
- LocalStorage save
- Smooth UI and animations

## 🚀 Getting Started
```bash
npm install
npm run dev
